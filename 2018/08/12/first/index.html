什么是Markdown

  markdown是一种纯文本格式标记语言。通过简单的标记语法，他可以是普通文本内容具有一定格式。
&emso; 在markdown中兼容HTML,但是有一些局限性，能在markdown中直接使用HTML标记
但有些标记需要制约。HTML区块元素不可随意在Markdown中使用，但是HTML的区段标签可以在markdown的段落、列表或者标题中随意是使用。例如：

This is an h1
=============
This is an h2
-------------

# h1
## h2
### h3
#### h4
##### h5
###### h6
